## `analyze.py`の実行

必要ライブラリ

```
pandas
matplotlib
```

```bash
# 環境の新規構築(最初の1回だけ)
$ python3 -m env venv

# 環境の有効化（検証環境に入るたびに実行）
$ source venv/bin/activate

# ライブラリのインストール (最初の1回だけ)
$ pip install matplotlib pandas

# プログラムの実行（検証環境に入るたびに実行）
$ python3 analyze.py

# 環境から抜ける（検証環境に入るたびに実行）
$ deactivate
```
